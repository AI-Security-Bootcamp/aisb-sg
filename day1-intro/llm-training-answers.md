An LLM is a machine learning model that uses stacked transformers to make next token predictions. Words are represented by tokens Transformers consist of the attention mechanism which allows conceptual understanding of the input referring to the whole text for prediction, and it is followed by a neural network at every layer allowing the non-linearity.

In pre-training phase the model is fed with massive free-text data and trained based on the next word prediction for every concurrent word in the text. After that during RLHF the model is given assistant capabilities. RLHF can also involve alignment work that can teach AI to prevent responding to harmful queries or include unwanted response content.
